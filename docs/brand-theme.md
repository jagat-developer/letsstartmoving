# Let's Start Moving Canada - Brand Theme

## Source Read

- Reference site: https://letsstartmoving.com/
- Canada site: https://letsstartmoving.ca/
- Core brand colors observed: deep red `#cf0402`, bright orange `#ff5a09`, charcoal `#1a1a1a`, white, light gray.
- Logo: skyline, mover, truck silhouette, red/orange/black treatment.
- Brand tone: reliable, local, transparent pricing, careful crews, fast quote response, licensed/insured confidence.

## Premium Direction

- Keep the red/orange brand DNA, but use it more selectively.
- Add warm ivory backgrounds, deep charcoal sections, brass-gold highlights, refined borders, large whitespace, and editorial service imagery.
- Use the Ottawa contact details and `.ca` service structure while borrowing the cleaner quote-first flow and trust proof from `.com`.
